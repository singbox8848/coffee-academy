## 第 1 步：创建 GitHub 仓库

1. 访问：**https://github.com/new**
2. 登录你的 GitHub 账号（没有就注册一个，免费）
3. 填写：
   - **Repository name**: `coffee-academy`
   - 选择 **Public**（公开）
4. 点击 **Create repository**

---

## 第 2 步：推送代码到 GitHub

创建仓库后，GitHub 会显示一行命令，类似：

```bash
git remote add origin https://github.com/你的用户名/coffee-academy.git
git push -u origin main
```

**在命令行运行这两句**（替换`你的用户名`为你的 GitHub 用户名）

或者复制粘贴：
```bash
cd C:\Users\52014\.openclaw\workspace\咖啡教学网站
git remote add origin https://github.com/你的用户名/coffee-academy.git
git push -u origin main
```

推送成功后，你的 GitHub 仓库里会有 3 个文件：
- ✅ index.html
- ✅ 部署说明.md
- ✅ Cloudflare 部署指南.md

---

## 第 3 步：连接 Cloudflare Pages

1. 访问：**https://pages.cloudflare.com**
2. 登录 Cloudflare 账号（没有就注册，免费）
3. 点击 **Create a project**
4. 选择 **Connect to Git**
5. 授权 Cloudflare 访问你的 GitHub
6. 选择 `coffee-academy` 仓库
7. 点击 **Begin setup**
8. 项目名保持 `coffee-academy`
9. 点击 **Save and Deploy**

---
