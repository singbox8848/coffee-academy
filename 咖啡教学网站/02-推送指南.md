# 📤 推送到 GitHub - 简单方法

## ⚠️ 需要认证

Git 推送需要登录 GitHub，有两种方法：

---

## 方法一：使用 GitHub Desktop（最简单 ⭐）

1. **下载 GitHub Desktop**
   - 访问：https://desktop.github.com
   - 下载并安装

2. **登录账号**
   - 打开 GitHub Desktop
   - 登录你的 GitHub（singbox8848）

3. **添加本地仓库**
   - File → Add Local Repository
   - 选择文件夹：`C:\Users\52014\.openclaw\workspace\咖啡教学网站`
   - 点击 Add Repository

4. **发布到 GitHub**
   - 点击右上角 "Publish repository"
   - Name: `coffee-academy`
   - 取消勾选 "Keep this code private"（要公开）
   - 点击 Publish

✅ 完成！现在访问 https://github.com/singbox8848/coffee-academy 确认文件已上传

---

## 方法二：使用命令行 + Personal Access Token

### 第 1 步：创建 Token

1. 访问：https://github.com/settings/tokens
2. 点击 **Generate new token (classic)**
3. 填写：
   - Note: `Coffee Academy Deploy`
   - Expiration: `No expiration`
   - Scopes: 勾选 **repo** (全选)
4. 点击 **Generate token**
5. **复制 Token**（只显示一次！）

### 第 2 步：推送代码

在命令行运行：
```bash
cd C:\Users\52014\.openclaw\workspace\咖啡教学网站
git remote add origin https://singbox8848:你的_TOKEN_在这里@github.com/singbox8848/coffee-academy.git
git push -u origin main
```

**注意：** 把 `你的_TOKEN_在这里` 替换成你刚复制的 Token

---

## 方法三：直接在 GitHub 创建仓库后手动上传

如果上面都觉得麻烦：

1. 访问：https://github.com/new
2. Repository name: `coffee-academy`
3. 点击 **Create repository**
4. 点击 **uploading an existing file**
5. 把 `index.html` 拖进去
6. 点击 **Commit changes**

⚠️ 缺点：这种方法不会上传部署说明文档
