## 🚀 永久部署方案

### 方案一：Vercel（推荐 ⭐）

最简单，一键部署，自动 HTTPS，全球 CDN。

**步骤：**

1. 访问 [vercel.com](https://vercel.com) 注册账号（可用 GitHub 登录）

2. 安装 Vercel CLI：
   ```bash
   npm install -g vercel
   ```

3. 在项目目录运行：
   ```bash
   cd 咖啡教学网站
   vercel
   ```

4. 按提示操作，完成后获得永久链接如：`https://your-site.vercel.app`

**优点：**
- ✅ 免费套餐足够用
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 绑定自定义域名

---

### 方案二：Netlify

同样简单，功能强大。

**步骤：**

1. 访问 [netlify.com](https://netlify.com) 注册账号

2. 安装 Netlify CLI：
   ```bash
   npm install -g netlify-cli
   ```

3. 在项目目录运行：
   ```bash
   netlify deploy --prod --dir=.
   ```

4. 获得永久链接如：`https://your-site.netlify.app`

**优点：**
- ✅ 免费套餐充足
- ✅ 表单处理功能
- ✅ 自动部署（连接 Git 后）

---

### 方案三：GitHub Pages

完全免费，适合已有 GitHub 账号的用户。

**步骤：**

1. 创建 GitHub 仓库（如 `coffee-academy`）

2. 将网站文件推送到仓库：
   ```bash
   cd 咖啡教学网站
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/coffee-academy.git
   git push -u origin main
   ```

3. 在 GitHub 仓库设置中：
   - 进入 Settings → Pages
   - Source 选择 `main` 分支
   - 保存后获得链接：`https://你的用户名.github.io/coffee-academy/`

**优点：**
- ✅ 完全免费
- ✅ 与代码版本管理集成
- ✅ 稳定可靠

---

### 方案四：Cloudflare Pages

新兴选择，性能优秀。

**步骤：**

1. 访问 [pages.cloudflare.com](https://pages.cloudflare.com)

2. 连接 GitHub 仓库或直接上传文件

3. 获得链接如：`https://your-site.pages.dev`

**优点：**
- ✅ 免费额度高
- ✅ Cloudflare 全球网络
- ✅ 快速部署
